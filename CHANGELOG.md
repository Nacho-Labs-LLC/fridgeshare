# Changelog

## 0.1.0 - 2026-04-30

Initial public release.

- Letter magnets with style presets
- Sticky notes with paper styles
- Photo uploads with captions
- Emoji magnets
- Dry-erase drawings
- Fridge surface themes
- Self-hosted board directory
- Admin token support for board management
- Real-time shared editing via revision polling
- Docker Compose deployment
- Image upload and asset serving API
- Backup-friendly file-based storage
